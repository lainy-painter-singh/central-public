# Deal Sharing Format

Each deal is one line: **Company Name — Description with metrics inline.**

## Style Rules

1. Always use em dash (—) between company name and description
2. Start with what the company does in plain language — no "Company X is developing..." or "Company X offers..."
3. Keep it concise: 1-3 sentences max
4. Weave metrics in naturally: revenue, ARR, ACV, growth rate, contracts
5. End with round info if known: "Series A." or "Raising $5-7M."
6. No founder names, LinkedIn URLs, or titles
7. No fluff phrases like "leveraging AI" or "positioned against competitors"
8. If data is sparse, keep it short — don't pad with speculation
9. Include company URL if available
10. Never say "sparse meeting notes" or "unable to generate" — write what you know, even if it's one sentence

## Example

Acme Corp — AI-powered supply chain optimization for mid-market manufacturers. $2M ARR, 3x YoY growth. Series A.

## Anti-Patterns (DO NOT generate these)

- "Sparse meeting notes provided, unable to generate a detailed summary." — NEVER
- "The company name suggests a focus on..." — NEVER speculate from the name
- "positioned against competitors like X and Y" — leave out competitor analysis
- Starting with "CompanyName is developing..." or "CompanyName offers..." — start with what they DO
